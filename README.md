Template-CLI to create projects as I would

## Usage

```bash
# Interactive mode
pnpm create gonzapp

# Override template
pnpm create gonzapp --template <template>

# Override name
pnpm create gonzapp --name <name>

# Get Command help
pnpm create gonzapp --help
```

## Installation

```bash
pnpm install -g create-gonzapp
```
